# 📌 CUSTOMER CHURN

## 🚀 Project Overview

This is a Machine Learning web application built using Streamlit that predicts whether a customer is likely to churn (leave) or stay.

The model is trained on telecom customer data and helps identify at-risk customers so businesses can take preventive action.

---

## 🧠 Problem Statement

Customer churn is a major issue in subscription-based businesses. The goal of this project is:

- Predict if a customer will churn
- Understand key reasons behind churn
- Provide actionable suggestions for retention

---

## ⚙️ Tech Stack

- Python  
- Pandas  
- Scikit-learn  
- Logistic Regression (ML Model)  
- Streamlit  
- Pickle  

---

## 📊 Features Used

The model uses customer attributes such as:

- Tenure (months)  
- Monthly Charges  
- Contract Type  
- Internet Service Type  
- Tech Support Availability  
- Other encoded service features  

---

## 🖥️ App Features

- Streamlit-based interactive UI  
- Real-time churn prediction  
- Probability score (churn risk %)  
- Automatic reasons for prediction  
- Personalized suggestions for customer retention  

---

## ▶️ Run Locally

Clone the repository:

```bash
git clone  https://github.com/logeshwarinaickar/churn-prediction-app.git
```

Go to project folder:

```bash
cd CUSTOMER-CHURN
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the app:

```bash
streamlit run app.py
```
